# Learning the new without losing the old

**Experiment AE · 2026-06-26 · qualified win**

The test Exp B couldn't run. We stream three registers that really differ,
Darwin's Victorian science, Shakespeare's Early-Modern verse, the King James
Bible's archaic scripture, in one pass, no replay, under bounded memory so a new
register forces eviction. A flat recency cache forgets a third of a bit on
Darwin; the brain-inspired stack (ECAN STI/LTI, CLS fast/slow, ART vigilance,
LIDA broadcast) forgets about 21× less (total backward +0.021 vs +0.454, Darwin
essentially flat) and has the better peak too (2.22 vs 2.52). The load-bearing
piece was ART resonance: reinforce the most-specific context that recognized the
input, not the most-salient generic one. It looked worse than baseline at first
and won two steps in. A bounded-memory phenomenon, the only regime a lifelong
learner lives in.

This experiment reads `darwin.txt`, `shakespeare.txt`, `bible.txt` from `data/`.

**Run it** (from the repo root, after `bash data/get-data.sh`):

```sh
python experiments/non-forgetting/run.py
```
