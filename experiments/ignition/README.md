# When the whole room agrees on a topic

**Experiment T · 2026-06-26 · qualified result**

Tests topic ignition: a higher level commits to one of 128 topics and broadcasts it down to every position; the lower predictor counts conditioned on local context AND the topic. Headline: it hurts at the character level (spelling does not depend on topic) but clearly helps at the word level exactly where local context has run out — and real topic beats a shuffled topic, confirming the machinery.

**Run it** (from the repo root, after `bash data/get-data.sh`):

```sh
python experiments/ignition/run.py
```
