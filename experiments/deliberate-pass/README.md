# Thinking slow, by counting

**Experiment AG · 2026-06-26 · win, with an honest negative**

A count-native System 2. A dual gate — calibrated confidence plus Botvinick
conflict — deploys a deliberate pass that overrides System 1 *only when it is
wrong*. The Engle signature is clean: on conflict cases (where the fast reflex is
wrong 88%) accuracy rises +0.38 (0.116 → 0.496); on no-conflict cases zero harm
(0.680 → 0.680); at zero step budget it falls back to System 1 bit for bit. The
honest negative: the elaborate serial workspace (focus, inhibition-of-return,
cognitive decoupling) does NOT beat a trivial "defer to the wider context" on this
one-step probe (0.15 vs 0.39 where they differ). The gate is the win; the
workspace is parked for the multi-step task.

**Run it** (from the repo root, after `bash data/get-data.sh`):

```sh
python experiments/deliberate-pass/run.py
```
