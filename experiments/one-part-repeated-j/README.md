# One part, repeated, wired bigger (J)

**Experiment IJ · 2026-06-25 · win**

The foundation/combiner study (Exp J) behind the uniform Column: shows multiplying experts raw makes the model fluent but wildly overconfident, and a calibrated geometric-mean pool (pool gently for scoring, sharpen only at generation) fixes cost while keeping fluency. One named, understood knob.
