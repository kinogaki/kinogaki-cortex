# You can't write your signature backwards

**Experiment V · 2026-06-26 · clear win**

Operationalizes the Thousand Brains trajectory-memory claim on text: is there a memory of CHANGE that transfers to unseen words, runs one direction only, and lets a cue prime what comes next? Headline: a model that keeps only vowel/consonant/space transitions degrades 4.5x less on unseen words than a spelling model (+25% vs +114%) — change transfers, content does not.

**Run it** (from the repo root, after `bash data/get-data.sh`):

```sh
python experiments/trajectory-memory/run.py
```
