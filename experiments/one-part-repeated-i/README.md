# One part, repeated, wired bigger (I)

**Experiment IJ · 2026-06-25 · win**

The uniform-Column build (Exp I): collapse the whole system to one part — a Column (an online counting predictor) — and show that wiring it wider (voting columns) and deeper (word then phrase level) gets better, attributably. Pairs with `one-part-repeated-j`. Headline: the calibrated geometric-mean pool is the hinge; the same part wired bigger improves cost and phrase coherence (81%→94%).

**Run it** (from the repo root, after `bash data/get-data.sh`):

```sh
python experiments/one-part-repeated-i/run.py
```
